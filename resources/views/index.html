URL Shortener

Create short URLs by sending a request to ->
http://url-shorter.herokuapp.com/new?url=%url%
(Where %url% is the url you would like to shorten.)

The resulting output looks like:
{"url":"http://example.com","short":"1gO"}

Optionally you can create URLs using this form:

Request ->

http://url-shorter.herokuapp.com/new?url=

Response ->

http://url-shorter.herokuapp.com/
